# Figure

The `<figure>` HTML element represents self-contained content, potentially with an optional caption, which is specified using the `<figcaption>` element. The figure, its caption, and its contents are referenced as a single unit.
(https://developer.mozilla.org/en-US/docs/Web/HTML/Element/figure)

## Default

We use `<figure>` together with image to improve accessibility with caption, just add the standard HTML5 figure code in the `<div id="qg-primary-content">` element, or add the class `qg-fig` to any figure element outside of the qg-primary-content element.

You can control the image description alignment left or right (turning it into a cut-in image) by adding the class `pull-right` or `pull-left` to the figure element `<figure class="pull-right">`, 
you also need to add `class="clearfix"` to the parent container.
